# AIT-724 Hands-On 2

[![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/ysenarath/AIT-724-HandsOn-2/blob/main/notebook.ipynb)

This repository contains the code and resources for Hands-On II of the AIT-724 course. The hands-on session focuses on using NetworkX for social network analysis and visualization.

The code is organized into a Jupyter Notebook (`notebook.ipynb`) that demonstrates various functionalities of NetworkX, including creating graphs, analyzing network properties, and visualizing social networks.
